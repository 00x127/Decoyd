Detection Methods

Read Detection
Detects when anything opens a bait file - cat, type, Notepad, Python, anything. Uses inotify on Linux, Windows Security Event Log (Event ID 4663) on Windows, atime polling as fallback.
Copy Detection
Watches Desktop, Documents, Downloads, Temp and cloud folders for new files whose SHA-256 hash matches any bait file. Catches cp, copy, drag-and-drop, anything.
